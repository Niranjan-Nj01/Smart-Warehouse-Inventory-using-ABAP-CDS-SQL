# Smart Warehouse Inventory using ABAP CDS and SQL

The **Smart Warehouse Inventory** project is a modern SAP ABAP application developed using the **SAP BTP ABAP Environment** and **Eclipse ADT**. It demonstrates the use of **Core Data Services (CDS)** and **ABAP SQL** to efficiently manage and analyze warehouse inventory data. The project joins material master and stock tables using CDS Views to calculate the total stock value and total quantity of each material across multiple warehouses. It also showcases SQL aggregation techniques using **SUM()**, **AVG()**, **GROUP BY**, and **HAVING** clauses to retrieve meaningful business insights, such as identifying materials with stock quantities greater than a specified threshold and calculating the average price per material. By leveraging modern ABAP syntax and database-centric processing, this project highlights best practices for building optimized, scalable, and maintainable SAP applications while demonstrating practical business scenarios commonly encountered in inventory and warehouse management systems.
